## 🛠️ 安装说明

1. 克隆此仓库到你的 Obsidian 仓库：
```bash
git clone https://github.com/你的用户名/obsidian_setup.git
```

2. 配置步骤：
   - 复制 `.obsidian` 文件夹到你的 Obsidian 仓库
   - 复制 `zob_config` 文件夹到你的仓库
   - 在 Obsidian 设置中启用所需的样式片段
   - 选择并应用你喜欢的主题

## 📁 目录结构

```
.
├── .obsidian/          # Obsidian 核心配置
│   ├── plugins/       # 插件配置
│   ├── themes/        # 主题文件
│   └── snippets/      # CSS 样式片段
└── zob_config/        # 自定义配置
    ├── js/           # JavaScript 脚本
    ├── template/     # 笔记模板
    └── css/          # 自定义样式
```
